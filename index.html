Informasi Baru untuk Pelanggan Disney+ Hotstar

Mulai 3 Oktober 2023, Disney+ Hotstar akan menawarkan dua Paket Langganan: Basic dan Premium, dengan pilihan langganan bulanan dan tahunan untuk masing-masing paket.

Paket Basic di Rp65.000 per bulan atau Rp450.000 per tahun dan Paket Premium di Rp119.000 per bulan atau Rp799.000 per tahun.

Bagi para pelanggan aktif, kunjungi tautan berikut untuk informasi terkait perubahan yang berlaku untuk paket berlangganan saat ini: https://h.hotstar.com/IDPricingAdvBi

Pilihan paket langganan terbaru ini diluncurkan sejalan dengan kurasi koleksi konten global serta konten original eksklusif yang semakin ekstensif untuk para pelanggan. Paket Langganan Basic dan Premium memberikan penawaran menarik dan fleksibilitas lebih bagi pelanggan. Infomasi lebih lanjut terkait paket, fitur dan harga, kunjungi: https://h.hotstar.com/SubPlanIDbi
